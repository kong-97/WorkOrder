# 车间生产看板系统

生产工单看板系统 - 供车间主任查看当日生产工单状态，并支持更新工单进度。

## 依赖

- Python 3.7+
- Flask

## 快速开始

### 1. 进入项目目录

```bash
cd WorkOrder/
```
### 2. 安装依赖

```bash
pip install flask
```
### 3. 运行项目

bash
```
python 面试代码-app_buggy.py
```
## 4. 访问系统
```
打开浏览器访问：http://127.0.0.1:5000
```
### 5. 功能说明
```
工单列表查看
工单进度更新（自动更新状态）
生产数据汇总看板
```
